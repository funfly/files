安装nodejs express mongo mongojs mongoose

1. 安装：node-v0.10.15-x86 到：C:\Program Files\ 下载地址：http://nodejs.org/download/
2. CMD->CD C:\Program Files\nodejs\node_modules\npm 
3. CMD->npm install
4. CMD->npm install -g express
5. 打开 Node.js command prompt 查看版本：
   CMD->express -V
   CMD->node -v
6. CMD->D:
7. CMD->mkdir nodejs
8. CMD->CD nodejs
9. 创建 express 项目：
   CMD->express --sessions --css stylus --ejs myapp
10.安装依赖模块
   CMD->CD myapp
   CMD->npm install
11.运行：
   CMD->node app
12.安装mongo mongojs mongoose
   CMD->npm install mongo
   CMD->npm install mongojs
   CMD->npm install mongoose
