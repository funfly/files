PhoneGap 初接触（andriod篇）

1. 下载并解压缩 Eclipse
   下载地址：http://pan.baidu.com/share/link?shareid=2151683953&uk=2670810502
   建议解压缩到：d:/phonegap/
   
2. 下载并安装 Android SDK
   下载地址：http://pan.baidu.com/share/link?shareid=2174930747&uk=2670810502
   安装 Android SDK 后，打开 Android SDK Manager 并安装 SDK 包：Tools 、Android 4.3 (API 18)、Extras
   
3. 运行 D:\phonegap\eclipse\eclipse.exe
   建议设置Workspace目录：D:\phonegap\workspace
   
   如有必要请下载并安装 jdk-7u25-windows-i586
   下载地址：http://pan.baidu.com/share/link?shareid=2170300315&uk=2670810502

4. 下载 ADT Plugin
   下载地址：http://pan.baidu.com/share/link?shareid=2202611859&uk=2670810502

5. 安装 ADT Plugin
   Eclipse -> Help -> Install New Software...
   
   重启 Eclipse

6. 下载并解压缩 phonegap-2.9.0.zip
   下载地址：http://pan.baidu.com/share/link?shareid=2215753850&uk=2670810502
   建议解压缩到：d:/phonegap/
